Elementos usados para criação de um site:

  • Para definir as fontes usadas: * { font-family: Arial, Helvetica, sans-serif; }
  • Para colocar gradiente no fundo: body { background-image: linear-gradient(to right, #F4C28D, #F29F8D); }
  • Para formatar um bloco de informações uso o Main e nele posso botar:
    • a cor do fundo: background-color: white;
    • uma borda: border-radius: 10px;
    • uma sombrinha e que defino na sequência qntos pixel para a direita, embaixo, espalhado e a cor: box-shadow: 5px 5px 15px #b95555a4;
    • definir a largura: width: 600px;
    • padding: 10px;
    • para centralizar: margin: auto;

Seletores Personalizados importantissimos no CSS

# é ID
. é Class
: é pseudo-classe (manipula dentro da class)
:: é pseudo-elemento (manipula o conteúdo de tudo)
> é children (filho, manipula o filho da class)

As informações do ID sempre sobrepõe as Class

Existe também as pseudos-classes, elas servem para definir estados de elementos.
Ex: div:hover{background-color....} aqui eu quero dizer que quando eu passar com o mouse nas divs do meu código vai mudar a cor do fundo.

DIV

E quando precisamos ter informações dentro de uma determinada Class e manipular essas info separadamente, vamos usar <div > p{....}
Aqui diz que estou manipulando um parágrafo que está dentro de uma div.

div.hover é pra quando eu quiser que algo aconteça quando eu passar o mouse.

Comandos pseudos-classes:

  • :hover para quando passar o mouse
  • :visited serve para link que já foi clicado
  • :active quando estiver clicando

Pseudo elemento

Ele mexe nas classes, elementos, id, ou seja, ele mexe diretamente no conteúdo.

Comandos aleatórios:

  • display:none serve para ocultar algo
  • display: block serve para mostrar
  • display: inline-block; pra dizer que quero tudo linha de bloco
  • height: 200px; definir altura
  • width: 200px; definir largura

  • background-color: cor de fundo
  • border: 1px solid rgb(69, 69, 69); define a espessura da barda, se ela é sólida e a cor
  • text-align: center; alinhamento do texto
  • color definir a cor
