ID e Class

Quando tenho vários elementos iguais no meu site e eu quero personalizar um específicio, eu uso o id, por exe: <h1 id=princial>, assim eu posso personalizar esse tílulo específico e não vai para os outros h1.

Quando preciso personalizar vários elementos iguais, ao invés de usar o ID, eu uso CLASS dentro do meu seletor, por ex: <h2 class ="piscar">, aqui eu digo que qualquer seletor que tiver a class piscar vai ter uma determinada configuração, que farei pelo CSS.

Importante: um ID só pode ser usado num seletor, já a CLASS posso usar ela em vários.

Tipos de Caixas:

Box-level (caixas que ocupam a largura toda do site, e sempre que vou inserir um conteúdo, ele vai quebrar linha e começar na próxima)
Exemplos de box-level: div, h1, p, main, header, nav, article, aside, footer, form, video.

Inline-level (caixas que se ajustam, se eu termino um parágrafo por ex, quando vou inserir outra coisa ele vai aparecer do lado, não tem quebra de linha como no box-level.)
Exemplos de inline-level: span, a, code, small, strong, em, sup, sub, label, button, input, select.
